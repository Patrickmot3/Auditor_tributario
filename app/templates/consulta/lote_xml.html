Importação de XML NF-e
Formatos aceitos: .xml (nota única), .zip ou .rar (várias notas)
Como funciona
  • XML único — processa uma NF-e por vez, exibe resultado item a item com CST sugerido.
  • ZIP com XMLs — importe um lote inteiro: basta compactar os arquivos .xml em um ZIP e enviar. Ideal para exportações do emissor.
  • RAR com XMLs — mesmo fluxo do ZIP. Requer que o servidor tenha o utilitário unrar instalado; se não disponível, use ZIP.
  • Compatible com NF-e versão 4.00. NCMs duplicados são atualizados, nunca duplicados no histórico.
